Acode Powerful Editor

This image is the representation of Acode tablet view. In a tablet acode appearance will be same as shown in above image.

Acode is a powerful code editor android app which is used for web development. It has snippets, previews, console, app view as well as browser view, and most important it host the previews in localhost very easily without any setup. It also has GitHub support and some other features like FTP. Just create a GitHub token with proper access settings and connect GitHub account with Acode. It will make easy to code and share.
